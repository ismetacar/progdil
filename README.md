# RandomName

RandomName dizini altında ProRandomAdjectiveClause.cs ve MainProRandomAdjectiveClause.cs

###derleme:
`$ gmcs -pkg:dotnet *.cs`
Bu şekilde derlendiği zaman MainProRandomAdjectveClause.exe oluşacaktır. 
###çalıştırma:
`$ mono MainProRandomAdjectiveClause.exe`
